# Personal Landing Page

Build a personal landing page that showcases who you are!

## Requirements

Your landing page must include:

### 1. Hero Section
- Your name displayed prominently
- A short tagline or bio (1-2 sentences)
- The hero section should have a `data-testid="hero"` attribute

### 2. About Section  
- A longer description about yourself (2-3 paragraphs)
- The about section should have a `data-testid="about"` attribute

### 3. Projects Section
- Display at least 2 projects you've worked on (can be fictional)
- Each project should have a title and description
- The projects section should have a `data-testid="projects"` attribute

### 4. Contact Section
- At least one way to contact you (email, social link, etc.)
- The contact section should have a `data-testid="contact"` attribute

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the dev server:
   ```bash
   npm run dev
   ```

3. Run tests to check your progress:
   ```bash
   npm test
   ```

## Tips

- Start with the HTML structure, then add styling
- Use semantic HTML elements (`<header>`, `<section>`, `<footer>`)
- Keep it simple - focus on clean, readable code
- Have fun and make it your own!

## Submission

Push your code to this repository. The automated tests will run and verify your submission.
